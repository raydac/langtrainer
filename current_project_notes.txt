## 1. Implementation Progress
- Null handling cleanup: resource module lists now normalize to empty lists; several internal "not found" helpers use `Optional` instead of returning `null`.
- RTL support: JSON `rtl` list, Hebrew keyboard layout, module/editor/Bricks/Crossword behavior, expanded `en-he-small` sample.
- External lessons: OS default folder, `LANGTRAINER_EXTERNALS`, GitHub sync, hardened sync paths, dual `index.json` merge, source icons, opened files shared across compatible modules.
- Resource selectors: `Start` / `Open from file` / `Sync Lessons` layout, tooltips, Start disabled until a row is selected.
- Virtual keyboard stays mouse-only to preserve typing focus; header row uses fixed control height.
- Crossword: rectangular board scaling; translation hint stays centered; dialog tip re-centers after text-direction updates.
- Gson replaced with Jackson; bundled classpath UI icons; optional JetBrains Mono with system monospace fallback.

## 2. Architecture & Technical Discoveries
- `DialogDefinition.modules()` is always non-null; an empty list means the resource is visible to all modules.
- `TextDirectionSupport` centralizes RTL orientation, alignment, and bidi embedding; RTL surfaces prefer `LangTrainerFonts.SYSTEM_MONOSPACED`.
- Crossword RTL: `placement.col()` anchors letter 0 (rightmost for horizontal RTL); re-apply `CENTER` on translation label after direction setup.
- Bricks RTL: mirrored build-strip margins; completion fly uses visible history width, not spacer width.
- External index merges `<externals>/common/jsons/index.json` and `<externals>/index.json`; entries may be absolute or relative to the index file.
- `GitHubFolderSynchronizer` uses anonymous GitHub API, rejects unsafe sync targets and non-portable paths; sync failures still show local resources when load succeeds.
- `InputEquivalenceSupport`: positional match when key/value sizes match, otherwise any-key; equal token sets act as equivalence classes.
- Build with `mvn22`; assembly descriptors need `**/*` includes; JSON via Jackson databind 2.21.
