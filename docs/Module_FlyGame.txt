Module FLY GAME
---------------

1. It is a child of AbstractLangTrainerModule
2. It is a dynamic interactive game
3. It works with resources described in ./JSON_format.txt
4. Load and select of such file is organized as in ./Module_Dialog.txt through list of resource file but in the case of the module from resource folder ./fly_game
5. The idea of the game
 5.1. It is a dynamic interactive game where user should learn new words from list for selected language.
 5.2. User sees blue field with seldom slowly moving white clouds from right to left
 5.3. For every learning word, the game starts a copter slowly flying from left corner to the right corner of field.

 5.4. User has time to enter correct word during time till the copter didn't reach space behind right field corner
 5.5. User drives the aim and position of aim relative target calculated through distance of entered word and expected word, distance in worse case must not be too big, and in correct input it should be just over the copter
 5.6. As only user entered word and pressed enter then we check that word is correct and if so then copter is exploded and we start learn next word, if word is not correct then do nothing
 5.7. If user didn't manage enter correct word till copter hidden behind the right corner then we show correct translation of word for 5 seconds and game begin again with the same word
 5.8. Words from source resource shown in random order till all them will be learned and correctly entered by user

6. Because application window can be scaled by user then the UI must be scalable and use SVG as source graphics. There must be two graphics buttons on the game field - to close the game and pause/resume
