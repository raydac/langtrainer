Module EDITOR
--------------

1. Child of AbstractLangTrainerModule; listed in Modules enum as EDITOR.
2. Swing UI to create and edit DialogDefinition-compatible JSON: metadata fields (title, optional menu path, optional module checkboxes, description, languages, optional RTL language list), a table of A/B line pairs with an optional Image column per row, and optional input equivalence rows (same semantics as JSON_format.txt). Path, modules, and RTL list are omitted from saved JSON when empty / none selected. Line images are edited via a button (thumbnail or "..."); dialog offers Load / Remove / Cancel, enforces the 64 KB limit from JSON_format.txt, and previews scaled to fit 400x400.
3. Load / save via file chooser (JSON). Not tied to index.json — it edits arbitrary paths the user picks; index.json remains the catalogue for bundled runtime lists in other modules.
4. Supports new document, font size controls for editor areas, and optional presets for English-style equivalence rows (see InputEquivalenceEnglishPresets in code).
5. Keyboard support follows AbstractLangTrainerModule#getSupportedLanguages (editor uses the same full virtual keyboard set as typing modules where applicable).
6. The A/B line table applies the optional RTL language list per column: column A follows Language A and column B follows Language B in both rendered cells and the active cell editor.
