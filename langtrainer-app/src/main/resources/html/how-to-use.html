How to use LangTrainer

  1. Choose a module on the main screen (Dialog, Fly game, Crossword, Editor).
  2. Open built-in content by selecting a list item, or load your own JSON file where supported.
  3. Start a session with Enter, Space, or mouse click.
  4. Type answers using your physical keyboard or the virtual keyboard button in module toolbar.
  5. Finish or return with module controls, then use X in the top-right to come back.

Tips

  • Use the Editor module to create or adjust your own training JSON packs.
  • In game modules, status lines and highlights explain progress and mistakes.
  • You can hide/show module top toolbar when module supports it.
