1.0.1 (SNAPSHOT)
 - Added loading of external JSON resources from a local externals folder.
 - Added "Sync GitHub" action to module resource selectors to sync public GitHub-hosted resources.
 - Added configurable externals folder path through LANGTRAINER_EXTERNALS, defaulting to ./externals.
 - Hardened GitHub folder sync for cross-platform paths and anonymous access.

1.0.0 (23-may-2026)
 - Initial desktop application for foreign-language training.
 - Added dialog practice, flying word game, crossword, and bricks training modules.
 - Added built-in editor for JSON language resources.
 - Added bundled sample resources and resource module filtering.
 - Added input equivalence rules for keyboard-friendly typing of accented letters.
 - Added Maven build, packaging, and application start configuration.
