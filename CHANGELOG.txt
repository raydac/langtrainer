1.1.0 (SNAPSHOT)
 - Added an Images training module for matching text bars to embedded line images, with cached card rendering for smoother play.
 - Added optional embedded images for JSON resource lines, with editor support, a 64 KB image limit, and hardened SVG decoding (no scripts/DTDs/external fetches; capped raster size).
 - Made Fly game practice less repetitive by reducing required repeat passes per word.
 - Fixed Fly game backspace doing nothing after a multi-word phrase was fully typed when the expected answer had trailing punctuation.
 - Added optional RTL language support in JSON resources, Hebrew virtual keyboard layout, and a bundled English-Hebrew sample.
 - Kept the virtual keyboard mouse-only so module typing focus is preserved while using it.
 - Added external JSON resources with local storage, `LANGTRAINER_EXTERNALS` override, bounded hosted-index sync with user-visible progress/failure messages, and opened files shared across compatible modules.
 - Clarified external sync results by reporting checked files separately from new, changed, and removed files.
 - Added `!MODULE` JSON resource markers so lessons can be hidden from specific module resource lists.
 - Added a Bricks help flash button and improved Unicode whitespace handling in phrase parsing.
 - Added a Bars matching module for dragging paired phrase bars into correct rows, with stage starts avoiding already matched rows.
 - Improved Bars readability by tinting left and right bars with distinct light backgrounds.
 - Improved Dialogs typing so punctuation is not auto-inserted while entering answers, mapped characters are still corrected, and minor typos can pass on Enter.
 - Improved resource list icons, folder labels, folder-first alphabetical resource ordering, crossword board scaling for wide puzzles, centered module hints, and crossword Home/End word navigation.
 - Replaced Gson with Jackson databind.

1.0.0 (23-may-2026)
 - Initial desktop application for foreign-language training.
 - Added dialog practice, flying word game, crossword, and bricks training modules.
 - Added built-in editor for JSON language resources.
 - Added bundled sample resources and resource module filtering.
 - Added input equivalence rules for keyboard-friendly typing of accented letters.
 - Added Maven build, packaging, and application start configuration.
